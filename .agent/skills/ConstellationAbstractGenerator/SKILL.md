# ConstellationAbstractGenerator

## Purpose

Produces publication-ready abstracts in two languages from the framing
fields generated by **FramingGeneratorMVP**.

> [!IMPORTANT]
> The Traditional Chinese abstract must be **written natively** in academic
> Traditional Chinese, not machine-translated from the English version.

## Input

| Field | Type | Source |
|---|---|---|
| `background` | `string` (1-2 sentences) | FramingGeneratorMVP |
| `purpose` | `string` (1-2 sentences) | FramingGeneratorMVP |
| `method` | `string` (1-2 sentences) | FramingGeneratorMVP |
| `result` | `string` (1-2 sentences) | FramingGeneratorMVP |
| `contribution` | `string` (1-2 sentences) | FramingGeneratorMVP |

## Processing Steps

1. **Build system prompt** — instruct the LLM to act as a bilingual academic
   writing assistant. Specify length targets and the native-writing constraint.

2. **Build user prompt** — pass the five framing fields verbatim.

3. **Call LLM** — request structured JSON output.

4. **Validate** — ensure both fields are present and within approximate length
   targets (±20 %).

## Output

```json
{
  "abstract_en": "~150-word English abstract …",
  "abstract_zh": "~150字正體中文摘要…"
}
```
